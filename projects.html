Dorm Room of Things

An Internet of Things platform for dorm rooms.
An Android application uses Bluetooth to connect to an Arduino controlled servo in order to control lights

Autonomous Moving Vehicle

Built an autonomous vehicle that follows a white line on a black table. As it moves, the vehicle emits noise depending on exposure to light

>>
This project was done in ECE 110, Introduction to Electronics.
The vehicle is controlled with an Arduino and common circuit devices. A line sensor breakout board has an IR emitting LED and an IR sensitive phototransistor.

Wolfram Demonstration

Effect of Friction on Ball Rolling Down a Ramp

A Wolfram Demonstrations Project is an interactive way to visually represent an idea. This project demonstrates the translational velocity of a ball as it moves down a ramp. The user can change the height of the ramp and whether or not friction occurs as the ball rolls.
